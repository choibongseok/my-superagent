### 1. **비동기 데이터베이스 처리**
```python
async with AsyncSessionLocal() as session:
    result = await session.execute(query)
    return result.scalars().all()
```
- SQLAlchemy의 AsyncSession 사용
- `run_async()` 헬퍼로 동기 Celery 태스크에서 async 함수 실행

### 2. **복잡한 SQL 쿼리**
```python
# Subquery: 각 사용자의 최근 태스크 날짜
subquery = select(
    Task.user_id,
    func.max(Task.created_at).label("last_task_at")
).group_by(Task.user_id).subquery()

# Main query: 비활성 사용자 필터링
select(User).outerjoin(subquery, ...).where(
    (subquery.c.last_task_at.is_(None)) |  # 태스크 없음
    (subquery.c.last_task_at < cutoff_date)  # 7일 이전
)
```

### 3. **주간 제한 계산**
```python
# 월요일 00:00 UTC 계산
week_start = now - timedelta(
    days=now.weekday(),
    hours=now.hour,
    minutes=now.minute,
    seconds=now.second,
    microseconds=now.microsecond
)
```
