### Idea #241: "Live Agent Collaboration Feed — 미션 컨트롤 뷰" 🎬📡

**날짜**: 2026-02-21 15:20 UTC
**우선순위**: 🔥🔥 CRITICAL-HIGH
**개발 기간**: 1일 (~100줄)
**AI 비용**: $0

**핵심 문제**:
- 현재 Task 실행 시 사용자가 보는 것: "⏳ 처리 중..." → (30초~2분 대기) → 결과
- **ChatGPT와 동일한 경험**: 로딩 → 결과. 멀티 에이전트의 장점이 전혀 안 보임
- Pipeline(#239)이 4개 에이전트를 순차 실행해도, 사용자 눈에는 "긴 로딩"일 뿐
- **"AI가 협업합니다"라는 말은 마케팅. 보여줘야 사실이 된다.**

**제안 솔루션**:
```
Task 실행 시 실시간 피드:
┌────────────────────────────────────────────┐
│ 🎯 "Q4 매출 분석 보고서"                    │
│                                            │
│ ✅ 14:02:01 Research Agent                 │
│    → 12개 소스 검색 완료, 8개 유효 데이터   │
│                                            │
│ 🔄 14:02:15 Sheets Agent (진행 중...)       │
│    → 데이터 3개 시트 정리 중                │
│    → 차트 2/3 생성 완료                     │
│                                            │
│ ⏳ 14:02:30 Docs Agent (대기)              │
│ ⏳ 14:02:30 Slides Agent (대기)            │
└────────────────────────────────────────────┘
```

**핵심 기능**:
1. **WebSocket Activity Stream** (~40줄): 기존 Celery task 이벤트를 WebSocket으로 전달
   - 이미 `celery_app.py`에 task 상태 추적 로직 존재
   - LangFuse trace 이벤트도 파이프 가능
2. **Agent Status Component** (~30줄): 각 에이전트의 현재 상태 + 요약을 실시간 표시
   - Pending → Running → Completed 상태 전환 애니메이션
   - 각 단계의 intermediate result 요약 (검색 결과 수, 생성된 차트 수 등)
3. **Pipeline Progress Bar** (~30줄): 전체 파이프라인 진행률 (Step 2/4)

**왜 이게 경쟁사와 완전히 다른가**:

| 경쟁사 | Task 실행 중 UX | AgentHQ + Activity Feed |
|--------|-----------------|------------------------|
| ChatGPT | "..." 점 세 개 | 에이전트 4개가 실시간 협업하는 장면 |
| Notion AI | 인라인 텍스트 스트리밍 | 리서치→분석→작성 전체 과정 가시화 |
| Google Duet | 앱별 개별 동작 | 앱 간 데이터 흐름 시각적 확인 |
| MS Copilot | "Working on it..." | 각 에이전트의 판단 과정 투명 공개 |

**이것이 AgentHQ의 킬러 UX다**: 사용자가 "이건 ChatGPT랑 근본적으로 다르구나"라고 느끼는 유일한 방법은 **여러 에이전트가 협업하는 장면을 직접 보여주는 것**이다.

**기술 기반**:
- WebSocket 재연결 로직 이미 구현됨 (Phase 3)
- Celery task state 추적 이미 존재 (`celery_app.py`)
- LangFuse tracing 연동 완료 → 에이전트별 step 이벤트 활용 가능

**Graduation Gate**: ✅ 100줄 이하, 1일, WebSocket 기반

**예상 임팩트**:
- 🎬 **데모 와우 팩터 10x**: Pipeline 실행이 "영화 같은 장면"이 됨
- 🧠 **멀티 에이전트 가치 즉각 체감**: 말이 아닌 시각적 증거
- 📊 **대기 시간 불만 해소**: 로딩이 아닌 "작업 과정 감상"으로 인식 전환
- 🏆 **경쟁사 완전 차별화**: 이 UX를 제공하는 AI 도구는 현재 없음

### Idea #242: "Output Portfolio & Smart Share — 바이럴 성장 엔진" 🔗🌍

**날짜**: 2026-02-21 15:20 UTC
**우선순위**: 🔥 HIGH
**개발 기간**: 1일 (~90줄)
**AI 비용**: $0

**핵심 문제**:
- Task 완료 후 결과물이 Google Drive에 생성되지만, **관리/공유 경로가 없음**
- 사용자가 "지난주에 만든 보고서 어디 있더라?"를 찾을 수 없음
- **가장 강력한 마케팅은 사용자가 직접 공유하는 것** — 그런데 공유할 수단이 없음
- 경쟁사(Notion AI, Canva AI) 모두 "Share" 버튼 → 바이럴 루프를 적극 활용

**제안 솔루션**:
```
[Output Portfolio 페이지]
┌────────────────────────────────────────────────┐
│ 📁 내 AI 산출물                                 │
│                                                │
│ 📊 Q4 매출 분석 보고서          2026-02-21      │
│    Docs · Sheets(3) · Slides    [🔗 Share]     │
│    by Research + Sheets + Docs Agent            │
│                                                │
│ 📈 경쟁사 분석                   2026-02-20      │
│    Docs · Sheets(2)             [🔗 Share]     │
│    by Research + Docs Agent                     │
│                                                │
│ 📝 주간 회의록                   2026-02-19      │
│    Docs                         [🔗 Share]     │
│    by Docs Agent                                │
└────────────────────────────────────────────────┘
```

**핵심 기능**:
1. **Output Gallery API** (~30줄): `GET /api/v1/outputs` — 완료된 Task의 산출물 목록
   - Google Drive 파일 링크, 생성 에이전트, 날짜, 타입별 그룹핑
   - 기존 Task 모델의 `result` 필드에서 Drive URL 추출
2. **Share Link Generator** (~30줄): `POST /api/v1/outputs/{id}/share`
   - Google Drive 공유 권한 자동 설정 (viewer)
   - 커스텀 공유 페이지: "AgentHQ로 만든 보고서입니다" + 미리보기 + CTA
3. **Output Gallery UI** (~30줄): Jinja2 또는 React 간단 갤러리 페이지
   - 카드 레이아웃, 필터 (타입/날짜/에이전트)

**바이럴 성장 메커니즘**:
```
사용자 A가 보고서 생성
  → 동료 B에게 공유 링크 전송
  → 동료 B: "이거 어떻게 만들었어?"
  → 공유 페이지 하단 CTA: "AgentHQ로 만들었습니다. 무료로 시작하기 →"
  → 동료 B가 가입
```

**왜 지금?**:
- Pipeline(#239) 실행 결과물이 4개 파일 → 이걸 한 곳에서 관리해야 함
- Demo(#237, #240) 후 가입한 사용자의 **리텐션** 장치가 필요
- 공유 링크는 **제로 비용 마케팅 채널** — 광고 없이 자연 확산

**경쟁 벤치마크**:
| 도구 | 산출물 관리 | 공유 | 바이럴 |
|------|------------|------|--------|
| Canva | ✅ 프로젝트 폴더 | ✅ 1-click | ✅ "Made with Canva" |
| Notion AI | ✅ 워크스페이스 | ✅ 페이지 공유 | ✅ "Notion" 브랜딩 |
| ChatGPT | ❌ 대화 기록만 | ⚠️ 대화 공유 | ❌ |
| AgentHQ | ❌ 없음 | ❌ | ❌ |
| **AgentHQ + #242** | ✅ 포트폴리오 | ✅ Share Hub | ✅ CTA 포함 |

**Graduation Gate**: ✅ 90줄 이하, 1일, 기존 Task 데이터 활용

**예상 임팩트**:
- 🔗 **바이럴 성장 엔진**: 모든 산출물이 마케팅 도구가 됨
- 📁 **리텐션 향상**: "내가 만든 것들" 포트폴리오 = 떠나기 아까운 가치
- 📈 **제로 비용 고객 획득**: 사용자가 알아서 전파
