### ✅ Fact Checker Enhancements (fact_checker_v2=True)

**New Features:**
1. **Wolfram Alpha Integration** 🧮
   - Automatic verification of numeric claims and calculations
   - Supports math expressions, population queries, distance calculations
   - Weighted contribution (30%) to overall confidence score
   - Graceful degradation if WOLFRAM_APP_ID not set

2. **Contradiction Detection** 🔍
   - Semantic analysis using GPT-4o-mini
   - Pairwise source comparison
   - Severity classification (high/medium/low)
   - Confidence penalty based on contradiction severity

3. **Weighted Confidence Scoring** ⚖️
   - Source quality weights reliability (0-1 scale)
   - Bonus for 5+ sources (+10)
   - Penalty for <3 sources (-10)
   - Contradiction penalties: high (-20), medium (-10), low (-5)
   - New thresholds: verified (≥85), likely (≥70), uncertain (≥50), unlikely (≥30), unverified (<30)

4. **Enhanced Source Quality Database** 📊
   - Expanded from 10 to 40+ predefined sources
   - Categories: Academic (90-95), Government (90-95), News (85-90), Educational (85-92)
   - Improved domain scoring: .edu (85), .gov (90), .org (70)
