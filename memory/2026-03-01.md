#### 1. ✅ **test_decompose_task_basic** - Pydantic Model Patching Issue

- **Affected**: 5 tests (test_apply_formatting, test_apply_named_style, test_insert_image, test_insert_page_break, test_create_bullet_list)
- **Error**: Expected 1 call to batchUpdate, got 2
- **Root Cause**: MagicMock chains create intermediate calls
- **Fix**: Create explicit mock for chain endpoint: `mock_batch_update = MagicMock(); mock_service.documents().batchUpdate.return_value = mock_batch_update`

#### 3. ✅ **test_init_basic** - ResearchAgent Attribute Error  
- **Error**: 'ResearchAgent' object has no attribute 'citations'
- **Root Cause**: 
  - API changed: `citations` attribute → `get_citations()` method
  - BaseAgent updated: `ConversationMemory` → `MemoryManager` (Phase 2)
- **Fix**: 
  - Changed `agent.citations` to `agent.get_citations()`
  - Updated type check from `ConversationMemory` to `MemoryManager`
